# 💪 健身打卡应用

一个为帮助你坚持健身而创建的 Web 应用！支持每日运动打卡和体重记录。

## 🚀 功能特性

- ✅ **每日打卡** - 记录你的日常运动
- ⚖️ **体重追踪** - 记录体重变化
- 📊 **数据统计** - 查看打卡次数和体重趋势
- 📋 **历史记录** - 查看所有打卡和体重记录
- 💾 **本地存储** - 数据保存在浏览器中

## 🛠️ 本地开发

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 `http://localhost:5173` 查看应用。

### 生成生产版本
```bash
npm build
```

## 🌐 部署到线上

### 选项 1: 使用 Vercel（推荐，最简单）

1. **注册 GitHub 账号**（如果还没有）

2. **将项目推上 GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/fitness-tracker.git
   git push -u origin main
   ```

3. **连接 Vercel**
   - 访问 [vercel.com](https://vercel.com)
   - 登录 GitHub
   - 点击 "New Project"
   - 选择 `fitness-tracker` 仓库
   - 点击 "Deploy"

4. **完成！** 你的应用会自动部署并获得一个可公开访问的 URL

### 选项 2: 使用其他平台

- **Netlify**: 类似 Vercel，同样简单
- **GitLab Pages/GitHub Pages**: 免费但需要更多配置
- **Heroku**: 更复杂，需要后端支持

## 📝 使用说明

### 打卡记录
1. 切换到"✅ 打卡"标签页
2. 输入或选择运动内容
3. 点击"打卡"按钮

### 体重记录
1. 切换到"⚖️ 体重"标签页
2. 输入体重
3. （可选）添加备注
4. 点击"记录"按钮

### 查看统计
- 在"📊 仪表板"标签页查看：
  - 今日打卡次数
  - 本周打卡总次数
  - 今日体重（如已记录）
  - 体重变化趋势

## 💡 建议

- 📅 建议每天固定时间打卡（如早上或晚上）
- ⚖️ 称重时间最好是早上刚起床时，没吃饭前
- 📊 不要过于关注体重的日常波动，关注周/月的趋势
- 🎯 设定合理的目标，循序渐进很重要

## 🔒 数据隐私

- 所有数据都存储在你的浏览器中（本地存储）
- 不会上传到任何服务器
- 清除浏览器数据会删除记录
